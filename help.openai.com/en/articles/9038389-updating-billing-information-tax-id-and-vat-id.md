# Updating billing information, Tax ID, and VAT ID

How to update billing information, Tax ID, and VAT ID for ChatGPT and OpenAI API accounts.

Updated: 2 days ago

# Overview

This article explains where to update billing information, including Tax ID and VAT ID, for ChatGPT and OpenAI API accounts. It also explains when billing changes apply to invoices.

If the Tax ID you provide is found to be invalid, it may be removed it from system records at any time and applicable taxes may be charged.

# Update billing information

How you update billing information depends on your account type.

## Individual ChatGPT accounts

1. Go to [chatgpt.com](https://chatgpt.com/).
2. Select your profile icon, then select **Settings**.
3. Select the **Account** tab.
4. Select **Payment**, then select **Manage**.
5. Update your billing information, including adding your Tax ID, as needed.

## ChatGPT Business accounts

1. Go to [chatgpt.com](https://chatgpt.com/).
2. Select your profile icon, then select **Workspace settings**.
3. Select **Billing**.
4. Select **Settings**.
5. Select **Edit**.
6. Update your billing information, including adding your Tax ID, as needed.

Note that only workspace owners can view the **Billing** tab.

## OpenAI API accounts

If you are an owner of an API organization, go to [platform.openai.com](https://platform.openai.com/), then select **Billing preferences** to view and update your organization billing preferences, including adding your Tax ID, as needed.

# Invoice changes

Changes to billing information or Tax ID apply to future invoices only. Previously issued invoices cannot be updated retroactively. If you need a review or possible reissue of a previously issued invoice, contact Support and include the relevant invoice number.
