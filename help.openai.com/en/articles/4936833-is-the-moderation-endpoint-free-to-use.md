# Is the Moderation endpoint free to use?

Cost of use for the Moderation endpoint

Updated: 3 days ago

Yes, the [Moderation endpoint](https://platform.openai.com/docs/api-reference/moderations) is free for OpenAI API users, and usage of this tool doesn't count towards your monthly [usage limits](https://beta.openai.com/account/billing/limits).  
  
To learn more see our [Moderation API guide](https://platform.openai.com/docs/guides/moderation/moderation).
