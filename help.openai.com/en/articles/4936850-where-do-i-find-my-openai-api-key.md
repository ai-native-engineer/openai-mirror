# Where do I find my OpenAI API Key?

Updated: 10 days ago

You can create and manage your API keys on the [API key page](https://platform.openai.com/api-keys). For security, a full secret key is only shown when you create it. If you lose it or did not save it, create a new key and update your application to use the new key.  
  
Check out our [Best Practices for API Key Safety](/en/articles/5112595) to learn how you can keep your API key protected.  
  
  
***Reminder: Do not share you API key with anyone!***
