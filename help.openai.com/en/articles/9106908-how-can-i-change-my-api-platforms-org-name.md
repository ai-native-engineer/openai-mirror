# How can I change my API platform's org name?

Changing your API platform's org name

Updated: 10 days ago

You can change the name of your org on the API platform by going to your [Organization settings page](https://platform.openai.com/account/organization) and changing the Organization name.

![API platform Organization settings with General selected and the Organization name field ready to edit](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-1c8069407900c97a0b74917a/d99d2551a70b0c9812347d26ebb88982/01.png)

This org name is a label for your org; the organization id is unique and cannot be changed.
