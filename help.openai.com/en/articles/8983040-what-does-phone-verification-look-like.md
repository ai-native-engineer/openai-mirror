# What does phone verification look like?

Updated: 14 days ago

**ChatGPT**

Phone verification is no longer required for new OpenAI account creation or ChatGPT usage.

**API**

Phone verification is now mandated on platform.openai.com for generating your initial API key, though not for any subsequent API key generation after that.

Our default drop-down is set to the United States.

![Image](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-a5b872caa203b21806467575/facacb5d1931cf4f415ed6702ed3671f/phone_verification_step_1.png)

If WhatsApp verification is available for the country you select, the screen may tell you that your one-time code will be sent to your number on WhatsApp.

![Phone verification form with India country code +91 and a WhatsApp yes or no prompt](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-2829c351d4e0f429f4eafaa8/657ec0430f6f6df88e8e47012a8d3e4d/phone_verification_step_2.png)

After the code is sent, enter the code from WhatsApp to continue.

![Phone verification form with India country code selected and WhatsApp delivery set to Yes](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-c051ae4660de8eeb8c62629e/f94fe32d81102ff356dc8c1818f858b4/phone_verification_step_3.png)
