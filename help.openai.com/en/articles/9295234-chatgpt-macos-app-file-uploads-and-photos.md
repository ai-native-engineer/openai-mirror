# ChatGPT macOS app - File Uploads and Photos

Uploading files and taking photos with the ChatGPT macOS app.

Updated: 10 days ago

The ChatGPT macOS app enables you to start new conversations with photos from your computer, or new photos you take.  
  
To get started, access the prompt window by opening the ChatGPT macOS app or opening the launcher. Click on the plus icon to access the photo tools.  
  
You can select Upload file to upload photos directly from your Photo library. You can also select Upload File to upload the photo or image from your file system instead.

![macOS Photos picker open in ChatGPT with the Cats album selected](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-c476cd8f2d0e95f4dea25752/0517b9da441e6fc593dcd851b25e7402/I6g2gzdMUZ4QvTxuT0YW9e5rgg81XLW1oST7N-GX8AEOh4xFOPEbJdUYxBjvZsgu_pTZraSyoCtVU2BlXMjj2HXgPOtg7cIuIsNTSKrZfq20w0mkRksAWskX7IVRk8qs)

Please note that you’ll need to grant the ChatGPT app access to your photo library to use this feature.

![macOS permission prompt asking to allow ChatGPT.app access to the Photo Library](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-7d1d41dff0ae12850eb465b4/a803a7746c99242e08eadfeb36afb97d/nNgdNrXQMDIdhr_Vgug5jYhlypm98h2ELiHsdDIbOTX67AJBxs1Bzg9lW67qA7jxpdkep-aBs0vjp37CAVRWrQpS-TmIzKXZLPaqipL19u12WLAx9D2uNbITWK5IsYRe)
