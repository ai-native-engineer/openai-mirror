# Data Export: Are my shared links included when I export my data from ChatGPT?

Are my shared links from ChatGPT included in my ChatGPT data export?

Updated: 3 days ago

Yes. Your export includes shared conversation link data in a `shared_conversations` file (ID, conversation ID, title, and anonymity setting).
