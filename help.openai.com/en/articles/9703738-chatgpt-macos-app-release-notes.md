# August 6, 2024

## Companion window

![ChatGPT macOS companion window rewriting an email draft into a more concise version beside the Mail compose window](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-374990b487f74b1ee99c00e1/866332b9f2450a841283fce85cd07e43/Companion_window_8_6.png)

The macOS desktop app now gives you side-by-side access to ChatGPT with the new companion window. It stays in front of all other windows so you can be more productive while working across ChatGPT and other apps.  
  
Use the Option + Space shortcut to

* Start a new chat
* Refocus the cursor to an open companion window
* Reopen the last companion window

Customize its position, reset time, and keyboard shortcut in Settings. You can also pop out a previous conversation into a companion window by clicking the icon from the main desktop app window.

## Fixes and improvements

### Keyboard shortcuts

* Added a new shortcut to stop streaming responses: Command + .
* Added a shortcut to quickly share the main desktop: Command + Shift + 1
* Added a shortcut to quickly share the active window: Command + Shift + 2

### Customization options

* Added an option to disable macOS autocorrect.
* Added a setting to show ChatGPT in the menu bar, dock, or both.

### Enhanced functionality

* Introduced horizontal scrolling for tables, code blocks, and mathematical formulas.
* Added support for @mentioning GPTs.
* Enabled the ability to choose which display to screenshot.
* Added a "Launch at Login" option, which is off by default.
