# ChatGPT macOS app - Screenshot Tool

A guide on taking screenshots with the ChatGPT macOS app.

Updated: 10 days ago

To get started, access the prompt window by opening the ChatGPT macOS app or opening the launcher. Click on the Plus icon to access the screenshot tool.

![ChatGPT macOS composer plus menu with Take Screenshot option](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-e1314aa390884a7e44dec5a6/ac351fade56b7018d314107dc0811615/image.png)

Hover your mouse over the Take Screenshot option to see all of the application windows that are currently open.  
  
You can also type the name of an open application directly in the launcher and select it to take a screenshot of an open window.

![ChatGPT macOS app search with Chrome typed and Take Screenshot for Google Chrome available](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-50896586643dda2527957982/b8e27e782935745b4e26a1ae172fb10d/fBdycU3sOWxZv7-aX9RDgffabtjJuD0iHR_pr9ye7AHXNeiC2iJnx1ZzsiPpMjSsXgW3IYW4TdczpuuCfW0UwRpQXjjooGJkOzCKVg3CitCoqpK5hRN84loE_f0mg2Le)

Please note that you’ll need to grant the ChatGPT app access to record this computer’s screen and audio to use this feature.

![macOS Screen Recording permission dialog prompting ChatGPT.app access in Privacy & Security settings](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-36b5ec64ebc3c2ac9f7e089b/344a9e01fef54214dee1c06863d3d6a0/YlkuIwc8-GGAf25nHx7NK37wvLV1D0EZe_1_s1HwBaSyoAXY55qGHbENk6K_1Cpu_rjHhNmpT-wPsx2Elow85ugLZL5LqSPXytlyh8eT4GbSr6FAr7FH9OymOCGF86eM)

These permissions can be enabled in your System Settings. You may need to restart the application for the updated setting to take effect.

![macOS Screen & System Audio Recording settings with ChatGPT access enabled](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-a50b1cbafe87e171afd4de58/f86a811a58441d07d1bcf9f897754bc7/WIRgpFLicVXokupr3plWRfwdGBOWeYQbsMQqIeEaG4LYQkqDh2rruozlOmztC_vLIDBp0JWPy7H3qFGNKPjK0k4ZVJhWALxfpY1zBge5iJhKutTSFQZkIVpZAVm5ob_N)

Once the screenshot has been attached, you can start a new conversation or continue a conversation with the screenshot.
