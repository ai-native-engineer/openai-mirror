# Enterprise compliance information for ChatGPT record mode

Updated: 3 days ago

## Transcription and file retention

* Audio files are deleted immediately after transcription
* Transcripts and summaries inherit workspace retention policies set by your admin.

## Compliance API

For Enterprise and Edu workspaces, canvases and transcripts created by record mode are captured by the Compliance API. Historical meetings imported into ChatGPT Meetings are not included in the Recordings Compliance API endpoints. Please refer to our [documentation](https://chatgpt.com/admin/api-reference#tag/Recordings) for additional information.
