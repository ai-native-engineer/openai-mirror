# Is API usage subject to any rate limits?

Updated: 2 days ago

Yes, all API usage is subject to rate limits. You can check out our updated guidance on rate limits and usage tiers [here](https://platform.openai.com/docs/guides/rate-limits).  
  
If you want to increase your limits, check out the [limits page](https://platform.openai.com/settings/organization/limits) in your settings and apply for an increase at the bottom of the page.
