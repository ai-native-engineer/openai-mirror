# How can I delete my API key?

Updated: 10 days ago

Deleting your API key is simple. Just head over to the [API key page](https://platform.openai.com/api-keys) and select the trashcan icon next to the API key you'd like to delete.

![API keys table with a delete trash icon for My Test Key](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-1c5cf264fc2fc5781a8ee2a5/df01216da167cafe4e1f5d4e2bf2f34f/Screenshot_2024-05-23_at_3_29_25_E2_80_AFPM.png)

**Please note** that deleted API keys cannot be used. This may halt any applications you have running using this API key.
