# How can I move my ChatGPT subscription to the API?

Updated: 3 days ago

We're excited that you're ready to explore our API! Our API service is billed and managed separately to ChatGPT. You'll be able to upgrade your API service to pay-as-you-go by adding a payment method in your API [account billing settings](https://platform.openai.com/settings/organization/billing/overview).  
  
Note that API usage is charged by the number of tokens used. For more information, please see our [pricing page](https://openai.com/api/pricing/).
