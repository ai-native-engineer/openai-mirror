# How to login to the Android app with Microsoft authentication

How to access the 'Continue with Microsoft Account' in the Android app.

Updated: 2 days ago

Requires app version **1.0.0022+.**

See [How to check the ChatGPT Android app version](/en/articles/8174832-how-to-check-the-chatgpt-android-app-version)

To access the Microsoft authentication option in the ChatGPT Android app follow these steps:  
  
Step 1: In the Android app login page click **Log in**

![Image](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-6d1a3a38973ab47da543d9ae/e92f8ec3db18942f367f16e27584402c/Screenshot_2023-07-25-08-50-13-271.jpg)

Note: The pictured Android login screen may differ from the current product experience. In the current Android app, select **Log in or sign up** to start sign-in, then follow the prompts to continue.

In the Web UI that opens click **Continue with Microsoft Account** to log in:

![OpenAI sign-in page on Android with Continue with Microsoft Account option](https://images.ctfassets.net/j22is2dtoxu1/intercom-img-f98702163a515cbf6857a7e2/5b84f180c743b55d848b5fad87178f00/Screenshot_2023-07-25-14-36-16-723.jpg)

Note: The pictured sign-in page may differ from the current product experience. The current Android sign-in flow may ask for your email or phone number first before showing available authentication options.
