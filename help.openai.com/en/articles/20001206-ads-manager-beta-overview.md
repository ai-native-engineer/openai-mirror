# Ads Manager Beta Overview

Create, launch, and manage campaigns in OpenAI Ads Manager Beta.

Updated: 10 days ago

OpenAI Ads Manager Beta is the platform for creating, launching, and managing campaigns in ChatGPT Ads, with tools to monitor performance and make updates in one place. This is a beta product and will continue to evolve over time.

## Ads Manager Availability by Country

Ads Manager Beta is currently available to advertisers in the United States, Australia, New Zealand, and Canada.

We’re looking to expand Ads Manager Beta availability to the United Kingdom in the coming weeks. Availability for additional countries and regions may expand over time.

## What you can do in Ads Manager Beta

Ads Manager Beta supports the core workflows required to run campaigns end to end:

* **Create and manage campaigns:** create campaigns individually in the guided workflow or create them at scale with bulk upload
* **Monitor performance:** track impressions, clicks, and spend across campaigns, ad groups, and ads using table views, charts, and CSV exports
* **Manage account settings:** access account-level settings such as members, permissions, [API keys](https://developers.openai.com/ads/api-overview), billing details, and change logs

Because Ads Manager Beta is still evolving, some capabilities are limited today and additional functionality will be added over time.
