# How to install the Work with Apps Visual Studio Code extension

Updated: 2 days ago

The Visual Studio Code extension is only for use with the ChatGPT macOS app Work with Apps feature. [Learn more about Work with Apps.](/en/articles/10119604-work-with-apps-on-macos)

*(Note: The extension also works with VSCode Insiders, VSCodium, Cursor, and Windsurf.)*

**For most VSCode forks, install the “openai.chatgpt” extension from the marketplace**

* For VSCode, use `vscode:extension/openai.chatgpt`.
* For VSCode Insiders, use `vscode-insiders:extension/openai.chatgpt`.
* For Cursor, use `cursor:extension/openai.chatgpt`.
* Or visit the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=openai.chatgpt) on web.

**For VSCodium and Windsurf, sideload a VSIX**

1. Download the extension: [openai-chatgpt.vsix](https://persistent.oaistatic.com/pair-with-ai/openai-chatgpt-latest.vsix)
2. In your IDE, open the Command Palette (CMD+Shift+P) and choose “Extensions: Install from VSIX…” and select the VSIX file

With the extension installed, ChatGPT can now work with files open in VSCode.  
  
  
[Learn more about Work with Apps.](/en/articles/10119604-work-with-apps-on-macos)
