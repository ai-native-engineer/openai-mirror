# What types of files are supported?

File uploads for ChatGPT

Updated: 10 days ago

Most common file extensions for text files, spreadsheets, presentations, and documents are supported, including XLSX, XLS, CSV, TSV, DOCX, PPTX, PDF, and TXT.

**Note** - we do not currently support `.gdoc`! Our recommendation instead would be to export these files as a `.pdf`, `.docx`, etc.
