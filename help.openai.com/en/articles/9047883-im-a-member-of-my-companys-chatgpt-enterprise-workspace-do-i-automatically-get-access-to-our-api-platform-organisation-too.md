# I'm a member of my company's ChatGPT Enterprise workspace. Do I automatically get access to our API Platform organisation too?

ChatGPT Enterprise workspace and API Platform org membership is managed separately

Updated: 2 days ago

## What's the difference between a ChatGPT Enterprise workspace and an API Platform organisation?

Your company's ChatGPT Enterprise workspace is a shared space in which you and your colleagues can use ChatGPT and its productivity features like Custom GPTs, image generation, and Canvas. Workspace access is usually managed by your company's IT admin.  
  
An API Platform organisation is a shared space where developers can create API keys, manage usage, and build custom solutions on top of our models.

## How does membership work?

**Membership of ChatGPT Enterprise workspaces and API Platform orgs is managed separately.** Being a member of a workspace for your company won't automatically mean you're a member of an organisation, and vice versa.  
  
Admins have a range of options for managing members, including [SSO](/en/articles/10468051-sso-overview), [SCIM](/en/articles/9627404-openai-chatgpt-scim-integration-faq), and using the membership pages to invite members directly.  
  
If you need to get access to an API Platform org or a ChatGPT Enterprise workspace, contact the administrator or owner at your company.
