# How can I stop my API subscription?

Learn how to cancel your API subscription, what happens after cancellation, and when your final charges will apply.

Updated: 3 days ago

You can cancel your API pay-as-you-go service at any time by following these steps:

1. Go to your [Billing Overview](https://platform.openai.com/account/billing/overview).
2. Click **Cancel plan**.

After you cancel:

* You will still receive a final invoice for any API usage incurred during the current billing month.
* After the final invoice is issued, your saved payment method will no longer be charged.
* You’ll still be able to access the platform with a free account, but API access will be paused unless you re-add a payment method.

**Note:** API billing is based on actual usage, so canceling your paid account does not refund any charges already accrued for the current month.
