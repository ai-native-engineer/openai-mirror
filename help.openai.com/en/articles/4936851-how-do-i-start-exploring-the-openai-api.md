# How do I start exploring the OpenAI API?

Resources to get started building with the OpenAI API

Updated: 10 days ago

The OpenAI API lets you build intelligent, multimodal applications across text, images, and audio, with a range of [model options](https://platform.openai.com/docs/models).  
  
The following links are great ways to get started:

* [Playground](https://platform.openai.com/playground): Experiment and iterate with different prompts and generations.
* [Developer quickstart](https://platform.openai.com/docs/quickstart): Simple building blocks for developers.
* [Help center](https://help.openai.com/): Find answers to frequently asked questions.
* [Example prompts](https://platform.openai.com/examples): See examples of how to build prompts

For hands-on developer support, go to the [OpenAI Community Forum](http://community.openai.com). The developer forum is a great place to communicate with other developers, learn best practices, ask questions about the API, and share your thoughts with the community! For news on the latest features in our API and events in our dev community, follow [@OpenAIDevs](https://x.com/openaidevs?lang=en) on X.
