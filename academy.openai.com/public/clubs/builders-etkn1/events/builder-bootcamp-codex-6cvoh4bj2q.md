# Advanced & Builder Skills

In this session, you’ll learn how to use Codex to plan, build, and ship real code changes in an existing repository. We’ll cover how to explore a codebase, use plan mode, define repo guidance with AGENTS.md, create reusable skills, and keep implementation scoped and reviewable.

You’ll also learn how to validate Codex-generated changes with linting, typechecks, manual smoke tests, and handoff notes. This session is geared toward builders who want to use Codex as a practical coding partner for faster iteration, safer implementation, and production-style development workflows.
