# Codex

Join us for a technical overview of Codex for administrators, IT, and security teams responsible for deploying AI software engineering agents across an organization. In this session, we’ll cover how Codex fits into the software development lifecycle, how teams can manage access through ChatGPT Enterprise workspace settings and RBAC, and how authentication, data privacy, auditability, and sandboxing work across local and cloud environments.

We’ll walk through practical rollout considerations, including approval modes, sandbox settings, managed policies, Team Config, repository standardization, MCP integrations, and reusable skills for team workflows. We’ll also review governance tools such as analytics dashboards, Analytics APIs, and Compliance APIs that help teams monitor adoption, impact, and agent activity. Designed for workspace owners, IT admins, security leaders, and platform teams evaluating how to deploy Codex safely and effectively at scale.
