## SUMMARY

In this video, you’ll learn how to use ChatGPT for real-time internet research and creating documents.

Here’s what you’ll learn:

* How to use the Search the Web feature to find live data and sources
* How to use Deep Research Mode for detailed reports and analysis
* How to plan, write, or code using Canvas Mode inside ChatGPT
