# K12 Teachers - Communicate

## Use Canvas in ChatGPT to shape family-facing drafts before you send them.

![Draft Family Communications In Canvas](https://cdn.gradual.com/images/https://d2xo500swnpgl1.cloudfront.net/uploads/oaiacademy/EDU-Content-Covers-23--203a19f8-4de6-4272-a909-7189c08eae87-1779409504998.jpeg?fit=scale-down&width=1200)

Canvas is a writing space inside ChatGPT. It is useful when you want to build, revise, and polish a longer piece of writing instead of getting one quick response in a chat.

For teachers, Canvas can help with family updates, attendance reminders, event explanations, classroom newsletters, and translated versions of approved messages.

## A Useful Workflow

You can use Canvas to:

Make the tone warmer or more direct

* Simplify the reading level

* Add missing context for families

* Translate the message for a family or caregiver

* Polish the final version before sending

* Download the result as a document when needed

## Example: Attendance Message

A teacher or school team could use Canvas to draft a clear explanation of an attendance policy.

Use Canvas for messages such as:

Weekly classroom updates

* Field trip reminders

* Attendance or homework expectations

* Conference follow-ups

* Event explanations

* Translated versions of reviewed school communication
