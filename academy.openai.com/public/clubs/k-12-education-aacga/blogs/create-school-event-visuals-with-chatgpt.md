# Create School Event Visuals With ChatGPT

# K12 Teachers - Communicate

## Use ChatGPT to create original visuals for school events, presentations, and classroom communications.

![Create School Event Visuals With ChatGPT](https://cdn.gradual.com/images/https://d2xo500swnpgl1.cloudfront.net/uploads/oaiacademy/create-school-event-visuals-cover-712ae987-c289-4463-86b8-fcfa1163a8aa-1780006239973.jpeg?fit=scale-down&width=1200)

ChatGPT can generate a new image from a description you provide. That can help when you need a visual for a school board deck, back-to-school flyer, family night announcement, or classroom handout r.

Start with the purpose of the visual, the setting, the people or objects that should appear, and the style you want. Then refine the image by giving feedback.

## When To Use It

* Create a welcoming image for a back-to-school night flyer.

* Add a school-specific visual to a presentation for families or a board meeting.

* Make a classroom activity sheet feel more connected to the current topic.

* Draft a visual for a school celebration, spirit day, or community event.

## What To Include In The Prompt

A strong image prompt usually includes these components:

|  |  |  |
| --- | --- | --- |
| **F**ield | What to include | Exampl**es** |
| Purpose | Where the image will be used | Newsletter, slide deck, flyer, hallway poster, family email |
| Setting | The location or environment | Classroom, school entrance, gym, library, cafeteria, outdoor campus, community space |
| People | Who should appear in the image | Students, teachers, families, staff, community members |
| Details | Specific context that makes the image useful | Season, event theme, activity, grade band, materials, school context |
| Style | The visual treatment | Photorealistic, hand-drawn, poster style, simple illustration, warm lighting, black-and-white printable |

## Refine The Image

Potential changes to ask for:

* Make it more like a rural school, urban school, or suburban school.

* Add warmer lighting or a sunset background.

* Make the style more hand-drawn, cartoony, or professional.

* Make the scene simpler so it works better on a flyer.

* Create a black-and-white version families can print or students can color.
