# K12 Teachers - Adapt Instruction

## Use ChatGPT to connect class learning to something happening now

![Turn Current Events Into A Classroom Mini-Lesson](https://cdn.gradual.com/images/https://d2xo500swnpgl1.cloudfront.net/uploads/oaiacademy/EDU-Content-Covers-26--c5053e1c-6c07-444d-8691-08b4c3d7ae05-1779411294579.jpeg?fit=scale-down&width=1200)

Current events can make lessons feel more relevant, but it takes time to find an appropriate story, simplify the language, prepare discussion questions, and explain the connection to families. ChatGPT can help assemble a short, reviewable mini-lesson.

This is especially useful for social studies, science, advisory, media literacy, or any class where students benefit from connecting ideas to the world around them.

### **Create Quick Outputs**

Ask ChatGPT to create a small classroom package:

* A student-friendly summary

* Three vocabulary words

* A five-minute lesson plan

* Discussion questions

* One bell-ringer or exit-ticket prompt

* A short family note

You can run this as needed, or set it up as a recurring task if you want a weekly current-events option.

### Try It

﻿[Try in ChatGPT](https://chatgpt.com/?q=Find%20one%20current%20event%20from%20this%20week%20that%20is%20appropriate%20for%20fifth%20graders%20and%20connected%20to%20%5Btopic%20or%20class%20theme%5D.%20Give%20me%20a%20student-friendly%20summary%2C%20three%20vocabulary%20words%2C%20a%20five-minute%20mini-lesson%2C%20three%20discussion%20questions%2C%20one%20quick%20bell-ringer%2C%20and%20a%20short%20family%20note%20explaining%20what%20we%20discussed.)﻿
