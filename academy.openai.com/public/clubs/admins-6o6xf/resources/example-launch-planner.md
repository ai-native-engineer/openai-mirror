# Launch Checklist

| Phase | Action Item | Owner | Resources |
| --- | --- | --- | --- |
| **Launch Planning** | Host internal launch planning meeting | Program champion | ﻿[Planning your rollout](https://academy.openai.com/home/clubs/admins-6o6xf/resources/planning-your-chatgpt-rollout)﻿ |
| ﻿ | Identify key roles: IT lead, functional leads, potential internal champions | Program champion | ﻿[Planning your rollout](https://academy.openai.com/home/clubs/admins-6o6xf/resources/planning-your-chatgpt-rollout)﻿ |
| ﻿ | Identify initial license recipients | Program champion + leadership | ﻿[Inviting and Managing Your Team](https://academy.openai.com/public/clubs/admins-6o6xf/resources/inviting-and-managing-your-team)﻿ |
| ﻿ | Send baseline survey | Program champion | ﻿[Template baseline survey](https://academy.openai.com/home/clubs/admins-6o6xf/resources/impact-survey-template)﻿ |
| **Workspace Setup** | Configure SSO, SCIM, and verify company domain | IT Admin | ﻿[Inviting and Managing Your Team](https://academy.openai.com/public/clubs/admins-6o6xf/resources/inviting-and-managing-your-team)﻿ |
| ﻿ | Adjust workspace-wide settings (connectors, sharing, data controls) | Program champion | ﻿[Feature Controls and Integrations with Your Tools](https://academy.openai.com/public/clubs/admins-6o6xf/resources/feature-controls-and-integrations-with-your-tools)﻿ |
| ﻿ | Provision ChatGPT Enterprise licenses and organize user groups | IT Admin/Program Champion | ﻿[Inviting and Managing Your Team](https://academy.openai.com/public/clubs/admins-6o6xf/resources/inviting-and-managing-your-team)﻿ |
| **End‑User Access & Enablement** | Publish internal "New User Hub" or wiki with links to resources or sample prompts | Program Champion | ﻿[Empowering and supporting your team](https://academy.openai.com/home/clubs/admins-6o6xf/resources/empowering-and-supporting-your-team)﻿ |
| ﻿ | Send welcome email with login info and onboarding resources | Program Champion | ﻿[Email templates for users](https://academy.openai.com/home/clubs/admins-6o6xf/resources/team-communication)﻿ |
| **Align AI to business objectives** | Engage with departments to discover and document use cases | Program champion + functional leads | ﻿[Empowering and Supporting Your Team](https://academy.openai.com/public/clubs/admins-6o6xf/resources/empowering-and-supporting-your-team); [Find and share use cases](https://academy.openai.com/home/clubs/champions-ecqup/resources/find-and-share-ai-use-cases-to-show-impact)﻿ |
| ﻿ | Align with departmental executives to identify additional use cases and enablement needs | Program Champion + Executives | ﻿[Planning your rollout](https://academy.openai.com/home/clubs/admins-6o6xf/resources/planning-your-chatgpt-rollout)﻿ |
| **Daily Habit Building** | Identify internal champions and create champions network | Program Champion | ﻿[Empowering and Supporting Your Team](https://academy.openai.com/public/clubs/admins-6o6xf/resources/empowering-and-supporting-your-team)﻿ |
| ﻿ | Set up Slack/Teams channel for use case sharing | Program champion | ﻿ |
| ﻿ | Host a use case showcase | Admin / Champions | ﻿[Use case showcase playbook](https://academy.openai.com/public/clubs/champions-ecqup/resources/use-case-showcase-playbook-2025-09-15)﻿ |
| ﻿ | Host a hackathon | Program champion | ﻿[Hackathon playbook](https://academy.openai.com/public/clubs/champions-ecqup/resources/hackathon-playbook-2025-09-15)﻿ |
| **Post‑Launch Measurement** | Conduct usage & adoption check-in (~30 days post-launch) | Program Champion | ﻿[ChatGPT user analytics guide](https://academy.openai.com/home/clubs/admins-6o6xf/resources/chatgpt-enterprise-user-analytics-guide)﻿ |
| ﻿ | Send internal impact survey (~45 days) | Program Champion | ﻿[Template impact survey](https://academy.openai.com/home/clubs/admins-6o6xf/resources/impact-survey-template)﻿ |
