# Codex

# Advanced & Builder Skills

Join OpenAI Academy for an introductory session on Codex, designed for anyone curious about building with AI—no technical experience required. We’ll start with a quick overview of what Codex is, key definitions, and how it works, before moving into live demonstrations of what you can create as a nontechnical user.

Through guided demos, you’ll see how Codex can support real-world use cases across work, everyday life, and creative projects—from automating tasks and generating tools to building things just for fun. We’ll close with dedicated time for Q&A, so you can explore how to apply what you’ve learned to your own goals.
