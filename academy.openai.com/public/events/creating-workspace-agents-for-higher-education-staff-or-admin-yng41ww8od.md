# Education

Join us for a practical OpenAI Academy webinar on how staff and administrators can identify, scope, build, test, and safely use Workspace Agents for recurring operational workflows. We’ll start with the basics: what Workspace Agents are, how they work, and when they are a better fit than a regular ChatGPT conversation or reusable skill. Then we’ll walk through how to choose a strong first use case, define the sources and review steps an agent needs, build a first version, and improve it through testing and feedback
