# Education

Learn how faculty, researchers, and university teams can create Workspace Agents that help with recurring academic and operational workflows. This session will show where agents are most valuable: when a task depends on trusted institutional knowledge, follows a repeatable process, or requires the same kind of judgment across many requests.
