# Beyond the Prompt: Moving from Conversation to Connected Workflows

Register

# Codex for Work

Many people started using ChatGPT by asking questions, drafting, summarizing, and exploring ideas. Those skills still matter, but AI is also becoming a way to take on larger pieces of work: gathering context, reasoning through steps, creating documents, and returning work that a person can review.

In this session, we'll show how AI-assisted work is moving from conversation toward delegation. You’ll learn how to define a useful outcome, provide the right context, set constraints and quality standards, and keep human judgment in the loop.

## Agenda

1. How AI-assisted work is evolving from conversation to delegation
2. How Codex illustrates a broader workflow pattern for delegated work
3. How skills, plugins, workspace agents, ChatGPT sites, and Codex can support repeatable work

## Who should attend

This session is designed for ChatGPT users who are comfortable with everyday prompting and want to start building more repeatable, reviewable ways of using AI at work.
