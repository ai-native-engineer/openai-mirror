# Building Websites with Codex Sites

Register

Have an idea you'd like to share online? Join us to learn how Codex Sites can help you create a website for a personal project, small business, community group, portfolio, or anything else you're passionate about.

This session is open to everyone and does not require any previous experience with coding or web design. We'll walk through the process together and show how AI can help make building and publishing a website more accessible than ever. Download Codex before the session to follow along: **<https://openai.com/codex>**.
