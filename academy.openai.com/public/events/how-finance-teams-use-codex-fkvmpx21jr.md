# Codex

Finance work runs on scattered context: close workbooks, forecast updates, KPI dashboards, prior MBRs, owner notes, and reporting packs.

Codex can help turn that context into tangible finance assets your team can review, refine, and share, no coding required. The key is knowing where to start.

Join us for **How finance teams use Codex**, a practical session on using Codex to build review-ready finance materials from real source inputs. We’ll walk through a monthly business review workflow and show how the same pattern can support variance analysis, reporting packs, model checks, and scenario planning.

In this webinar, we’ll cover:

1. What Codex can help finance teams build
2. A walkthrough building a monthly business review narrative
3. How to review Codex output before sharing
4. Where to find starter prompts and finance use cases to try

Whether you’re preparing leadership reviews, refreshing reporting materials, or explaining variance drivers, this webinar will help you choose a first finance workflow and start building!
