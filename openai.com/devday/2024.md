Explore guides to start building

[### Realtime API

Developers can now build fast speech-to-speech experiences into their applications.

Learn more](http://platform.openai.com/docs/guides/realtime)

[### Vision Fine-Tuning

Developers can now fine-tune GPT-4o with images and text to improve vision capabilities.

Learn more](http://platform.openai.com/docs/guides/fine-tuning/vision)

[### Prompt Caching

Get automatic discounts on inputs that the model has recently seen.

Learn more](http://platform.openai.com/docs/guides/prompt-caching)

[### Distillation

Fine-tune a cost-efficient model with the outputs of a large frontier model.

Learn more](http://platform.openai.com/docs/guides/distillation)
