# Codex for Open Source

[Codex for Open Source⁠(opens in a new window)](https://developers.openai.com/codex/codex-for-oss) is a program to support the maintainers behind critical open-source software. 

Maintainers carry significant responsibility by reviewing pull requests, triaging issues, maintaining releases, and preserving security and code quality across widely used projects.

This program helps  reduce coding and review load with tools built for real maintenance workflows.

Selected maintainers receive:

* 6 months of ChatGPT Pro, which includes Codex
* Conditional access to Codex Security
* API credits for coding, maintainer automation, release workflows, and core open source work
