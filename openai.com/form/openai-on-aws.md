## Get access to OpenAI on AWS

Soon, customers will be able to access OpenAI frontier intelligence, Codex, and Bedrock Managed Agents on the trusted AWS platform where businesses already operate.

What’s available in limited preview:

* **OpenAI models on AWS:** Access OpenAI models within the AWS services, security controls, and procurement workflows you already use.
* **Codex on AWS:** Use Codex through Amazon Bedrock with AWS-native security, billing, and infrastructure.
* **Bedrock Managed Agents:** Build agents that can use tools, handle multi-step workflows, and take action in production environments.
