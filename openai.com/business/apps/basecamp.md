## How it works

Let ChatGPT securely access Basecamp projects, message boards, and to-dos to answer questions and summarize work using live Basecamp content.

### Get instant project visibility

Ask questions about completed, overdue, or assigned to-dos across Basecamp projects and receive clear answers using synced project data.

### Summarize project activity

Summarize recent message board posts and to-do activity to understand what’s happening across Basecamp projects.

### Reduce context switching

Reference Basecamp project context in ChatGPT to answer questions and review next steps without opening multiple tools.

## Sample prompts

Identify key risks and mitigation actions

**@Basecamp** Identify the top 10 risks across active projects based on overdue items, unanswered questions, and stalled threads; keep it blunt.

Summarize project status

**@Basecamp** I joined mid-project. Give me the current status for [project] including goals, latest decisions, open questions, and next milestones.

List my assigned tasks and deadlines

**@Basecamp** Pull all to-dos assigned to me across projects, group them by urgency, and highlight dependencies or blocked items.
