## How it works

Connect Jam screen recordings to ChatGPT to pull structured debugging context—including video, user events, logs, errors, and network requests—for understanding and organizing product issues.

### Inspect recordings with full context

View Jam recordings with associated video, user events, console logs, errors, and network requests to understand what happened in a session.

### Explain issues and reproduction steps

Use structured data from a Jam to explain what went wrong and generate clear reproduction steps based on exact user actions.

### Create engineering-ready artifacts

Turn Jam recordings into engineering-ready bug tickets, summaries of customer impact, test cases, or grouped issue themes using structured traces.

## Sample prompts

Inspect recording with full context

**@Jam** Open this Jam recording: [link]. Summarize what happened using the video, user events, console logs, errors, and network requests.

Generate clear reproduction steps

**@Jam** Based on this Jam recording, write precise reproduction steps using the exact user actions and environment details shown in the trace.

Draft engineering-ready bug ticket

**@Jam** Turn this Jam recording into a bug ticket. Include title, impact, environment, repro steps, expected vs. actual, suspected area (based on logs/network), and attach the key error snippets.
