# Get started with the Codex Security plugin

Choose the setup that fits how you work. Use Desktop Codex for a guided flow, or Codex CLI for a one-command scan from a folder of code.

Step 1

### Install Codex

If Codex is already installed and you’re signed in, skip to the next step.

[Install Codex(opens in a new window)](https://developers.openai.com/codex/quickstart?setup=app)

![Codex Security plugin card with the Add plugin button highlighted.](https://images.ctfassets.net/kftzwdyauwt9/FHniMRguyEWtHDL63ZBuU/e39beeb6d7886c3e6baad1ba787ac1f5/install_codex.png?w=3840&q=90&fm=webp)

Step 2

### Add the Codex Security plugin

Add the plugin in Codex. After installation is complete, the button changes to “Try in chat.”

[Add Codex Security Plugin(opens in a new window)](https://chatgpt.com/plugins/share/676aca3811d54fa7bcdef5255236b3c4)

![Codex Security plugin shown as installed with a checkmark.](https://images.ctfassets.net/kftzwdyauwt9/5h2XYxDhPLw5qBjmI4AZ95/29e432aa032c4e2df9868065a7fd082e/add_codex_security_plugin.png?w=3840&q=90&fm=webp)

Step 3

### Click “Try in chat”

This starts a new Codex chat with a Codex Security scan prompt ready to run.

![Codex chat composer prefilled with a Codex Security scan prompt.](https://images.ctfassets.net/kftzwdyauwt9/4CR547bl2PLuK5k4DV8jvf/47117e40ad63331c28efd05a1e84d743/try_in_chat.png?w=3840&q=90&fm=webp)

Step 4

### Pick a project folder

Choose a folder that contains the code you want Codex Security to review.

![Project picker showing a project called Repository selected in Codex.](https://images.ctfassets.net/kftzwdyauwt9/3Ck3ewHlkJLXnOi1IepUlb/b67f06389fe720235d0ceb9d77ded35c/select_folder__1_.png?w=3840&q=90&fm=webp)

Step 5

### Send the scan prompt

Codex opens with the Codex Security scan prompt ready. Press “Send” to start the scan.

![Codex composer send button highlighted to run the scan.](https://images.ctfassets.net/kftzwdyauwt9/4P7SG5JP5vY2ebj2wuahpb/19c1ec71f313f3b3d3d840420a142ee0/run_scan.png?w=3840&q=90&fm=webp)
