# OpenAI licenses GPT‑3 technology to Microsoft

OpenAI has agreed to license GPT‑3 to Microsoft for their own products and services.

![An impressionistic landscape showing a grassy hillside with scattered trees under a pastel sky blending pink, purple, and yellow hues.](https://images.ctfassets.net/kftzwdyauwt9/7ABCRUdfaIGxgR29t6WMAD/fd5bd12c35d0f204c4ca8d27a91ccf9f/openai-licenses-gpt-3-technology-to-microsoft.jpg?w=3840&q=90&fm=webp)

OpenAI released its first commercial product back in June: an [API⁠](/index/openai-api/) for developers to access advanced technologies for building new applications and services. The API features a powerful general purpose language model, [GPT‑3⁠(opens in a new window)](https://arxiv.org/abs/2005.14165), and has received tens of thousands of applications to date.

In addition to offering GPT‑3 and future models via the OpenAI API, and as part of a [multiyear partnership⁠(opens in a new window)](https://news.microsoft.com/2019/07/22/openai-forms-exclusive-computing-partnership-with-microsoft-to-build-new-azure-ai-supercomputing-technologies/) announced last year, OpenAI has agreed to license GPT‑3 to Microsoft for their own products and services. The deal has no impact on continued access to the GPT‑3 model through OpenAI’s API, and existing and future users of it will continue building applications with our API as usual.

Unlike most AI systems which are designed for one use-case, OpenAI’s API today provides a general-purpose “text in, text out” interface, allowing users to try it on virtually any English language task. GPT‑3 is the most powerful model behind the API today, with 175 billion parameters. There are several other models available via the API today, as well as other technologies and filters that allow developers to customize GPT‑3 and other language models for their own use.
