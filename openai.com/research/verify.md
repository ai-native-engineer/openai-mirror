# Verify OpenAI-generated images

Upload an image to check for signals that it was generated with OpenAI tools.

#### What does this tool do?

This tool checks whether an uploaded image contains provenance signals associated with images generated by OpenAI tools. It looks for supported signals, including C2PA metadata and SynthID watermarks, and reports whether either is detected.

![icon depicting a wrench](https://images.ctfassets.net/kftzwdyauwt9/5rRtPBCQYoLN5poacJhfqp/ead486284c437b6863a00541434c114d/settings-wrench.svg?w=3840&q=90)

#### What images can it detect?

The tool is designed to detect images generated with ChatGPT, the OpenAI API, or Codex. Another image can be uploaded, but OpenAI generation signals will only be detected if the image was generated with our tools.

![icon depicting sharing](https://images.ctfassets.net/kftzwdyauwt9/1gkOgwgs05hyxFzTRPomPe/a2f1cc014915c212ac61e16ba205426b/share-ios.svg?w=3840&q=90)

#### How do I use it?

Upload a single image, then review the results to see whether the tool detects C2PA metadata, a SynthID watermark, or no supported signal. For best results, crop screenshots closely around the image and avoid uploading files that contain multiple images.
