Description

This demo lets you talk to a 3D solar system with Realtime speech-to-speech and trigger animations with tool calling. Try asking about different planets, ask questions and see animations get triggered. For example, ask how much land there is on Earth versus water, which volcanoes are the highest on Mars, or what are Jupiter's moons.
