Description

Swifty Roguelike is a playable native macOS prototype created with Codex and GPT-5.5. It wraps a classic ASCII roguelike board in a SwiftUI Liquid Glass interface, with procedural rooms, corridors, doors, stairs, water, foliage, turn-based movement, bump-to-attack combat, loot, inventory, leveling, and descendable dungeon floors.

Use $imagegen and @Build macOS apps to build a small native macOS roguelike in SwiftUI. Start by creating the mood and UI direction, then turn it into a real playable app.

### Iterations

1. Use $imagegen to generate a screenshot-style mood image for the initial native macOS roguelike idea.
2. Use @Build macOS apps to build a small SwiftUI game that feels like a classic ASCII dungeon crawler inside a modern Liquid Glass macOS interface.
3. Include a procedural dungeon, turn-based movement, monsters, bump-to-attack combat, loot, potions, leveling, inventory, and a combat log.
4. Polish the user interface and ensure the first map is explorable and playable end to end.
5. Final step

   A playable Swifty Roguelike prototype, packaged with a screenshot and creation-story README for GitHub.
