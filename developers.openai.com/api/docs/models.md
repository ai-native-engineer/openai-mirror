# Models

## Choosing a model

If you're not sure where to start, use [GPT-5.5](/api/docs/models/gpt-5.5), our flagship model for complex reasoning and coding. If you're optimizing for latency and cost, choose a smaller variant like [GPT-5.4 mini](/api/docs/models/gpt-5.4-mini) or [GPT-5.4 nano](/api/docs/models/gpt-5.4-nano).

All latest OpenAI models support text and image input, text output, multilingual capabilities, and vision. Models are available via the [Responses API](/api/docs/api-reference/responses) and our [Client SDKs](/api/docs/libraries).
