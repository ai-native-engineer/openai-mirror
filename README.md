# openai-mirror

OpenAI 공개 자료, OpenAI Academy, 개발자 문서를 **마크다운으로 정리한 비공식 아카이브**입니다.
강의/자료 제작 시 참고 소스로 쓰려고 모았고, 누구나 읽어볼 수 있게 공개해 둡니다.

> ⚠️ **비공식(Unofficial) 아카이브입니다.** OpenAI가 만들거나 운영하지 않습니다.
> 모든 콘텐츠의 저작권은 **OpenAI**에 있으며, 이 저장소는 원문을 마크다운으로 변환해 보관한 개인 참고용 미러입니다.
> 최신/정확한 내용은 항상 아래 공식 출처를 확인하세요.

## 무엇이 담겨 있나

크롤 시점(스냅샷) 기준 텍스트 본문만 마크다운으로 보관합니다. 이미지/동영상 파일은 포함하지 않습니다(영상은 자막을 텍스트로 전사).

| 경로 | 내용 | 문서 수 |
|---|---|---|
| `openai.com/` | 뉴스, 연구, index 기사, 정책, global-affairs, 제품 페이지 | ~1370 |
| `developers.openai.com/` | API 레퍼런스, 가이드, cookbook, codex 문서 | ~3410 |
| `academy.openai.com/` | OpenAI Academy 블로그/리소스/이벤트 + 영상 자막 전사 | ~550 |
| `help.openai.com/` | 헬프센터 article | ~450 |
| `model-spec.openai.com/` | OpenAI Model Spec 전문 | 1 |
| `openaifoundation.org/` | OpenAI Foundation | ~10 |
| `openai.fund/` | OpenAI Startup Fund | ~3 |

`academy.openai.com/public/videos/`의 영상 레슨은 Vimeo 자동생성 자막을 텍스트로 전사해 보관합니다.

## 출처 (Sources)

- https://openai.com (news, research, index, policies)
- https://developers.openai.com (API, cookbook, codex)
- https://academy.openai.com (OpenAI Academy)
- https://help.openai.com (Help Center)
- https://model-spec.openai.com (Model Spec)
- https://openaifoundation.org , https://openai.fund

## 범위 밖

제품 앱과 운영/사용자 콘텐츠는 아카이브 목적 밖이라 제외합니다: chatgpt.com, sora(제품 앱), openai.fm(데모), community.openai.com(사용자 포럼), status.openai.com(운영 상태).

## 갱신

`openai-mirror` 스킬로 위 출처를 다시 크롤해 스냅샷을 덮어씁니다. 변경 이력은 git이 보존하므로 `git log` / `git diff`로 변경분을 추적합니다(날짜별 디렉토리를 누적하지 않습니다).

## 라이선스 / 저작권

이 저장소의 텍스트 콘텐츠 저작권은 전부 OpenAI에 있습니다. 학습/참고 목적의 미러이며, 저작권자의 요청이 있으면 해당 자료를 내립니다.
